# Responsible Use

All probabilities are estimates, not certainties.

The first scaffold uses mock and seed data. It must not be used as a real betting recommendation engine.

Before production use, the system needs:

- Historical match and odds data.
- Backtesting on Euro 2024, Copa America 2024, and World Cup qualifying data.
- Calibration reports.
- Provider terms-of-service review.
- Clear bankroll and risk controls.

Current local outputs may flag positive expected value from seed odds. Those flags prove the pipeline works; they do not prove a real betting edge.
